**GovSocial** is a modern web application designed to bridge the gap between citizens and government officials. It provides a transparent, "heavenly" user experience for reporting community issues and tracking their resolution in real-time.

## ğŸŒŸ Key Features

* **Dual-Role Authentication**: Secure login system with distinct interfaces for **Citizens** (to report) and **Government Officials** (to resolve).
* **Visual Reporting**: Citizens can describe issues in detail and attach photos to provide clear evidence.
* **Community Support System**: A "Support" (Like) feature allows other users to upvote critical issues, helping officials prioritize high-impact complaints.
* **Resolution Proof**: Government officials can upload proof of resolution (images/URLs) once a task is completed.
* **Impact Hub**: A live dashboard tracking the total number of reports and successfully resolved cases.
* **Cloud Persistence**: Integrated with **Firebase** to ensure data is accessible across any device, moving away from local-only storage.
